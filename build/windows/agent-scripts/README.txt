IG E-Sign USB Agent
====================

1. In the IG E-Sign portal, open USB Agent and click Download for Windows.
2. Run IG-E-Sign-Agent-Setup.exe to install (portal.url is included automatically).
3. Launch IG E-Sign Agent — pair from the app window with a code from the portal.
4. Close the window to keep the agent in the system tray; reopen it from the tray menu.
5. The agent runs in the Windows system tray (near the clock). Right-click the icon for status and Quit.
6. Insert your USB DSC token when signing from the portal.
