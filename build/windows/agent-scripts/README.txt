IG E-Sign USB Agent
====================

1. In the IG E-Sign portal, open USB Agent and click Download (or use the ZIP).
2. If you downloaded from the portal, copy portal.url into this folder.
3. Run "Pair Agent.bat" and enter the pairing code from the portal.
4. Run "Start Agent.bat" before using USB Sign in the portal.
5. Insert your USB DSC token when signing (PKCS#11 support coming soon).
