IG E-Sign USB Agent
====================

1. In the IG E-Sign portal, open USB Agent and click Download (or use the ZIP).
2. If you downloaded from the portal, copy portal.url into this folder.
3. Run "Pair Agent.bat" and enter the pairing code from the portal.
4. Start IG E-Sign Agent from the Start menu, desktop shortcut, or "Start Agent.bat".
5. The agent runs in the Windows system tray (near the clock). Right-click the icon for status and Quit.
6. Insert your USB DSC token when signing from the portal.
